“Quarks are very small particles…”

Quark is a “do-it-yourself” framework designed to create lightweight applications by using the power of your system’s webview.

Install Quark ⇩

Installing and using Quark could never be easier, add this to the dependencies section of your Cargo.toml

[dependencies]
libquark = "4"

Minimal Size

The size of a normal Quark application can be little as 178KB.

Powered by Rust

Paired with a high-performance backend made in Rust, it's made for speed and reliability.

It just works*

Simple, straightforward, and reliable. No complex configurations or setups needed.

Low dependency

Low dependency count makes rapid build cycles have less wait time.
